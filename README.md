# Mi Proyecto Portafolio
# Ruben Senra Perez


## Descripción de la práctica
Esta práctica consiste en el desarrollo de un portafolio web personal utilizando estándares de HTML y CSS. El objetivo es demostrar las habilidades adquiridas en la estructuración de documentos web y el despliegue de proyectos en plataformas como GitHub.

## Tecnologías utilizadas
* **HTML5**: Estructura semántica del contenido.
* **CSS3**: Diseño, maquetación y estilos visuales.
* **Git/GitHub**: Control de versiones y despliegue.
*  **Gemini**: para el porcentaje use la ia
